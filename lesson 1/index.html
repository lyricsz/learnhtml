Main Content

HTML is the basic building block of every website. It is the foundation of all websites (other technologies build upon it). HTML is the language mainly used for building plain website (no styling, interaction, dynamicity).

HTML Stands for HyperText Markup Language. This further clarifies that HTML is not a programming language but a markup language. Markup include text, paragraphs, images, list, tables e.t.c.

The "HyperText" in HTML simply is the text that when clicked changes the URL (URL stands for Uniform Resource Locator -- also known as web address ) to the HyperLink. It's fine if you've not grasp what HyperText means yet. Let's break it down further.

Have you ever visited a website where you click on the "About" and it takes you to another webpage (About Page)? The text "About" is the HyperText and the new URL (or address) -- "About Page" is the HyperLink.

Note:-

The HyperText is always visible whereas the HyperLink is not visible. HyperText is text that when clicked helped us navigate to another page. I'll explain this further later.

Pre-requisite in order to learn HTML.

In order to learn HTML, you only need to have the passion, a good computer, a web browser (most recommended - Chrome), and a text editor / program editor software if you are familiar with it.

To help with the text editor / program editor (free software), check the below list:

Android Text Editors (Not recommended to learn coding / programming on Android) :-
  1. Acode: A Very Powerful text editor with a live preview of your HTML codes. It is much more recommended after it's recent updates.
  2. SpkEditor: This editor is not fully offline. But what i personally enjoy about it is being able to add libraries (for programmers) to one's project easily and fast.
  3. AIDE: Not 100% recommended since it's not 100% free. But this program editor allows you to write your HTML codes and test it for free.

    4. Other editors include TrebEdit, WebCode, Code Studio e.t.c

Editors for PCs:
  1. Notepad: This is a free text editor that probably comes pre-installed (already installed) on most PC's (either MAC OS, Window, or Linux). It is not that recommended for professional use, but i'll recommend it for beginners
  2. VS Code: Visual Studio Code is a lightweight program editor. It is lightweight but very powerful program editor. It also allows you to add extensions (more functionality) from Extension Store.
  3. Sublime Text: Sublime Text is another free option out there. It's been out for long time, and still powerful till date. It is widely used by professional programmers.
  4. Intellij Ide: Although not free, this is a very powerful program editor. It includes so many features and it can be used for other cool stuff apart from web development. It has a free version online (mainly created for students) .

    5. Other editors include, Notepad++, Visual Studio e.t.c.
