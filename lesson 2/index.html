Course Overview

What is HTML? What is it used for? Is HTML a programming languge? Why is it important to HTML? This chapter introduces you to HTML Afresh.

Main Content

As discussed in the previous chapter, HTML stands for HyperText Markup Language. We also discussed what HyperText and Markup is.

HTML is the basic building block of every website. HTML handles our websites presentation for us (with ease). Styling and Dynamicity can't be done using HTML, we need two more language (CSS and JavaScript). This is known as the front-end (we'll clarify it more later).

The latest version of HTML is HTML5. It is maintained by the W3C (World Wide Web Consortium). HTML has transitioned over the years of the first version released. Back in the days we can't use HTML to create Web Games (programmers use Flash), watch movie (without the help of a plugin), play music and lot more. HTML5 made this available to us.

HTML is a markup langauge created by Tim-Berners Lee. When the Internet was first created (WWW) by Tim-Berners Lee, user's can't navigate from a web page to another (web pages are not linked together). To access a webpage, we must enter the URL (or address) manually. This is a lot of headache for web users. HTML fixes this by allowing users to navigate around the web just by clicking HyperText.

Here are the versions of HTML over the years (after it's first created):

Version Released date
HTML 1991
HTML 2.0 1995
HTML 3.2 1997
HTML 4.01 1999
XHTML 1.0 2000
HTML 5 2008-2014

The HTML5 version was first recommended by the WHATWG in 2008, but it become standardize on 2014 by the W3C. The W3C are currently working on standardizing some new version of HTML (HTML 5.1 and HTML 5.2). You can learn more about the W3C at their official website.

To further explain what fornt-end is, and it's associated technologies (HTML5, CSS3, JavaScript - ES9), HTML is basically like the human-skeleton of our web pages, CSS is like the skin, JavaScript is the live that makes it breath. Let's take a look at below definitions:-

HTML
I am the presentation of your webpage, the structure of your webpage.
CSS
I style the look and feel of your webpage, making it beautiful and more appealing.
JavaScript
I make your whole page dynamic and interactive (^-^)
What is dynamicity?

You might've notice the word "Dynamicity" ealier, this simply means your web page looks different across all users. E.g Amazon looks absolutely different from a user to another user. Static website is just it's opposite. What i see is exactly what other sees. (Web page visitors are often called traffic).

Before we learn about the pros and cons of learning the HTML langauge, let's discuss about the front-end and the back-end

The front-end is basically what you see on the webpages. However the back-end is like a powerhouse that makes webpage available and accessible.

More about Tim-Berners Lee

Timothy John Berners-Lee is the creator of the WWW(World Wide Web), HTTP(HyperText Transfer Protocol), and HTML.

The HTTP is what makes the communication between a server and a client (traffic) possible. Have you ever wondered why all webpages start with either http or https? Well, now you know. We'll discuss more about this and how to put your website online in ahead chapters...

Advantages and Disadvantages

The Advantages of learning HTML5 is listed below:

  1. HTML is very easy to learn.
  2. HTML is the basic building block of every webpages. So, learning it is essential.
  3. HTML with other web technologies like CSS and JavaScript pave way for front-end jobs.

The Disadvantages of learning HTML5:

  1. Learning HTML, there is only little to no job available for us.
  2. HTML alone is so boring, static and less appealing.
