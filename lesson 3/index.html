Hello World! And Welcome to the most fun chapter you've ever seen so far.

First, We already know HTML. But A Webpage is made up of element, and element consist of tags and contents. We'll expand on this later.

HTML code consists of only two things: DOCTYPE and ELEMENTS. So, Whatever we see in our webpage is an element? But is that 100% true? Let's find out using an example:

index.html
<⊕Copy>
<!DOCTYPE html>
<html lang="en-US">
<head>
<title> John Doe Greetings </title>
</head>
<body>
<p> Happy New Month </p>
</body>
</html>

This code might look a bit cryptic but we'll explain what it does later. For now, run the above code.

Element VS. Tags VS. Content

An Element in a HTML code consist of tags and content. The basic syntax for a tag is as below:

<tagName> & </tagName>

The above syntax might be hard to deciphered. Let's help with that. A basic HTML tag will have opening tag and closing tag. The above syntax can also be written as below:

<openingTag> & </closingTag>

Notice that we have a less-than symbol, followed by the tagName, followed by the greater-than symbol. (For the opening tag). The only difference between an opening tag and a closing tag is the slash character in the closing tag ( / ).

Look at our above example, Take a look if you can find the following tags:

<title> & </title>
<body> & </body>

If you did, congrats!!! you are getting what tags mean. If you don't I assure you will before we end this chapter.

Check out below table:

The Tags Meaning
<!DOCTYPE html> The doctype declaration, tells the browser which version of HTML you're working on. This doctype declaration (<!DOCTYPE html>) tells the broswer the code was written using HTML5.
<html lang="en-US"> This tells the browser "hey , our html codes start after". (It's like a hint for the web browser to read what's inside it as HTML codes). This is the opening HTML tag.
<head> The opening head tag. Every webpage is divided into two sections. The head sections are always invisible to the end-user. However we used them to add information about our web page. (e.g the title, the character set)
<title> John Doe Greetings </title> The title element. The content is the Title that appears at the top bar of our webpage. Here, the content is "John Doe Greetings".
</head> Closing head tag.
<body> Opening body tag. The body section is what's visible in our webpage. This is where our paragraphs, headings, list, images, videos e.t.c goes.
<p> Happy New Month </p> The paragraph element. The paragraaph element displays the content as a paragraph in our webpage. The content here is "Happy New Month".
</body> Closing Body tag.
</html> The closing HTML Tag.
Note:-

Opening tag and closing tag is also known as Starting tag (Opening tag) and Ending tag (closing tag).

I'll encourage you get more familiar with the above table. It explains our Earlier code. But... Wait! You notice we use content and element? What is that? What is DOCTYPE Declaration?

HTML Element

Now that we know what HTML tag is, let's discuss about HTML Element. HTML element consist of HTML Tag and It's content.

Note:- <html> & </html>

The HTML Tag is often refer to as the root tag. To avoid confusing it with other Tags... That we all known as HTML tag.

A content of a HTML Tag is what's inbetween the Opening tag and the Closing tag. Take a look at below code:

<tagName> Content </tagName>

Take a look at the word Content in the above code, you can easily notice it's inbetween Opeing tage and Closing tag? So... That is... You got it right. That is the content. Take a look of our Earlier HTML Code, Look for the below code:

<p> Happy New Month </p>

The opening tag is <p>, the closing tag is </p>, and the content is Happy New Month. Kindly note that the Content can also be an HTML Element. But what is HTML Element?

HTML Element consistof Starting tag, Ending tag, And it's content. Let's use an example to know it more better:

<p> I AM A PARAGRAPH </p>

From the above code, we can say the Tag in it is <p> & </p>, the content is I AM A PARAGRAPH. But the combination of both the tag and the content is the Element in the above code, So the above code is an entire HTML Element (IT consist of opening and closing tag and the content).

I believe by now we know what HTML Tag is, What content is, and What an HTML Element is? You can make a request in our WhatsApp group if by now you still haven't grasp it yet. (It's very important to know this).

Note:-

If a content of an HTML Element is Also another HTML Element, then we can say that the Element is a nested element. We also have empty element. They only have Opening tag with no closing tag. We'll explain that better in the next chapter.

DOCTYPE Declaration

The Document type Declaration (also known as DOCTYPE Declaration) is what sets our HTML Code apart. We don't want to let the browser guess what version of HTML We are using. What if we're using HTML5 and the browser guess we are using HTML4? The DOCTYPE Declaration is what comes first in your HTML Code. It lets the browser knows which version of HTML Your code is.

Each version of HTML has it's specialized DOCTYPE Declaration. For the previous versions before HTML5, The DOCTYPE was so cryptic to know and be able to remember so quick. However, HTML5 Provides us much easier example. Take a look at HTML5 DOCTYPE Declaration.

<!DOCTYPE html>

It's way easier to know and remember. Let's take a look at HTML4 doctype declaration:

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

Ha! This seems cryptic, isn't it? Well, Thanks to HTML5 for making life easier for us.

Always include DOCTYPE declaration in your HTML code.

A quick re-explanation is .... anything between the Opening Head tag and the Closing Head tag (it's content) will not be displayed in our webpage. Except anything inbetween the Opening Body Tag and the Closing Body Tag .
